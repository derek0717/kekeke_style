kekeke 批踢踢風格

一按把kekeke換成黑色風格

bug回報
  • 利用使用者定義腳本在kekeke加入按鈕，一按換成黑色風格
  • chrome / firefox適用
  • 開放源碼
  • 懂html / javascript的同學可以自行修改

步驟1: 安裝附加元件

  • 要運行「使用者定義腳本」先需要一個chrome / firefox附加元件；有安裝過adblock等插件的同學應該不陌生
  • chrome: 到Chrome程式商店找Tampermonkey （Tampermonkey）
  • firefox: 到附加元件找Greasemonkey（Greasemonkey）
ChromeFireFox
  • 這兩個猴子插件並不是我做的，而它們本身並沒有任何功能，用途是在指定網頁上運行用家自訂的腳本，來加入自訂功能
  • 猴子插件安裝完成後（有可能需要重啟瀏覽器）就可以加入kekeke用的腳本了

步驟2: 加入腳本

  1. 安裝好猴子插件後，每逢見到~.user.js字尾的使用者定義腳本路徑，都可以自動進入安裝確認頁面，非常方便
  2. kekeke用的腳本我已經掛在GitHub上，點一下進行安裝（blackeke.user.js）
  3. 不放心的同學可以先到GitHub上自行查看代碼，然後再點RAW按鈕安裝：gist.github.com/dragonairex/37aedc6c5a0344a52a6ecec0153a15ad
ChromeFireFox

步驟3: 使用

  • 安裝好後在kekeke按f5，就會出現菜單上的K按鈕。接下來應該不用說明了

4. FAQ

Q:怎樣刪除 / 修改腳本?
A:在猴子的管理頁面可以啟用 / 停用各個自訂腳本，查看腳本的運行情況 / 版本，亦可修改 / 刪除甚至新增自訂腳本
ChromeFireFox
Q:怎樣刪除Tampermonkey / Greasemonkey?
A:跟所有附加元件一樣，在瀏覽器的附加元件管理頁面進行
Q:我有bug要回報 / 想增加功能
A:可以點這裡到GitHub的bug回報頁面，或是到kekeke白貓版找loy也行（最近不是經常上）
